AssistLoop Chat Widget

Integrate AI-powered customer support into your Odoo website.

Features

  • Easy Setup - Configure in minutes through Website Settings
  • AI-Powered - Leverage AssistLoop's AI agents for instant customer support
  • Customizable - Position the widget on left or right side
  • Human Handoff - Seamlessly escalate to human agents when needed

Configuration

  1. Go to Website → Configuration → Settings
  2. Find "AssistLoop Chat Widget" section
  3. Enter your Agent ID from AssistLoop dashboard
  4. Enable the widget and save

Requirements

  • Valid AssistLoop account
  • Configured AI agent in AssistLoop

For support, visit assistloop.ai
