# AssistLoop AI Chatbot for Odoo

Integrate the AssistLoop AI-powered customer support chatbot into your Odoo website.

## Features

- Easy configuration through Website Settings
- AI-powered customer support 24/7
- Human handoff when needed

## Installation

### Odoo Apps Store (Recommended)

1. Go to **Apps** in your Odoo instance
2. Search for **"AssistLoop AI Chatbot"**
3. Click **Install**

### Manual Installation

1. Clone this repo to your Odoo addons directory
2. Restart Odoo and update apps list
3. Install "AssistLoop AI Chatbot"

## Configuration

1. Go to **Website → Configuration → Settings**
2. Find the **AssistLoop AI Chatbot** section
3. Enter your **Agent ID** from [AssistLoop Dashboard](https://assistloop.ai/dashboard)
4. Enable the chatbot and save

## Requirements

- Odoo 14.0, 15.0, 16.0, 17.0, 18.0, or 19.0
- Valid AssistLoop account
